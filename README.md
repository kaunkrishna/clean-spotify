# CLEAN SPOTIFY
Your music, zero distractions, without the clutter.

## Install

### Install the "Stylus" Extension in your browser

### Click on the button below to install the userstyle
[![Install Style](https://img.shields.io/badge/Install-Stylus-blue?style=for-the-badge)](https://raw.githubusercontent.com/kaunkrishna/spotify-clean-ui/main/userstyle/clean-spotify.user.css)

#### NOTE
* Spotify uses dynamic class names - may break after updates
* Use an ad blocker, please (uBlock Origin recommended)
